How to Run XPath Utility (Portable)
====================================

1. Unzip the deployment folder anywhere on your Windows PC.
2. Double-click 'start_app.bat' to launch the application.
   - The app will set up everything automatically.
   - Your browser will open the UI.
3. To stop, close the command window.

Requirements:
- Windows 10/11
- Edge browser installed
- Internet connection (first run only, for Python package installation)

No technical knowledge required. If you see any error, contact your admin.
